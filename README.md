# Machine-Learning-Linear-Regression-for-Starters
These are the codes I use to enhance my understanding for machine learning, specifically linear regression. Some may be simple, but good for beginners to use.
The Links to Code file contains the links to the code in python about training a model that predicts home prices with factors: area of the house, distance to subways, distance to sea, and the floor of the house. The MSE may be high (around 1700) because I purposely turn the noise in price high (scale with 30 in python np.random.normal(). Feel free to change that and see.)
